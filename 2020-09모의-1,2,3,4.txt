Coupling
DSCECC
Data -> Stamp -> Control -> Extern -> Common -> Content

Data : 자료 요소(call-by-value)
Stamp : 자료 구조
Control : 제어 변수
Extern : 외부 변수
Common : 전역 데이터(call-by-reference)
Content : 내부기능 및 내부자료

Coincidental : 서로간에 어떠한 의미있는 연관관계도 지니지 않은 기능 요소로 구성되는 경우이며, 상위 모듈에 의해 호출되어 처리상의 연관성이 없는 서로 다른 기능을 수행하는 경우.
