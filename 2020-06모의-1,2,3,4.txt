Data Coupling -> Stamp Coupling -> Control Coupling -> Extern Coupling -> Common Coupling -> Content Coupling

Coincidental Cohesion -> Logical Cohesion -> Temporal Cohesion -> Procedure Cohesion -> Communication Cohesion -> Sequential Cohesion -> Functional Cohesion 

Thread
한개의 프로세스가 여러개의 스레드를 가진다면 -> Multi Thread
커널 스레드의 경우 운영체제에의해 스레드를 운용
사용자 스레드의 경우 사용자가 만든 라이브러리를 사용하여 스레드를 운용.
스레드를 사용함으로써 하드웨어, 운영체제의 성능과 응용 프로그램의 처리율을 향상시킬 수 있음.
